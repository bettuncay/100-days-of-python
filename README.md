# 100-days-of-python
 This repository is a collection of daily Python projects completed as part of the "100 Days of Python" challenge. Each day focuses on a new topic or concept, ranging from basic programming skills to advanced techniques.  Throughout this journey, I aim to strengthen my Python programming knowledge, improve problem-solving skills, and build a portfolio of diverse projects.  Structure:  Each day is organized into its own folder (e.g., Day-1, Day-2), containing the code and a README file explaining the project or exercise. Topics covered include data structures, algorithms, object-oriented programming, web development, and more! Feel free to explore the projects, provide feedback, or even try the challenge yourself.
